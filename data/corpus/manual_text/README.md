# Manual Text Overrides

Use this folder for private, reviewed text/markdown replacements when a source
file extracts poorly but should remain cited under its original Blackboard,
Rencai, or transcript path.

The override path mirrors the original source path and appends `.md` or `.txt`.
Examples:

- `data/corpus/manual_text/rencai/Visa in China/Staying in China/Living in China/Renting an Apartment in China.doc.md`
- `data/corpus/manual_text/blackboard/Some Scanned File.pdf.md`

The app repo ignores override contents because they may contain student-resource
material. Keep the actual override files local or in private storage.
